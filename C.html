C Programming - Introduction

C programming is a general-purpose, procedural, imperative computer programming language developed in 1972 by Dennis M. Ritchie at the Bell Telephone Laboratories to develop the UNIX operating system. C is the most widely used computer language. It keeps fluctuating at number one scale of popularity along with Java programming language, which is also equally popular and most widely used among modern software programmers.

In 1978, Brian Kernighan and Dennis Ritchie produced the first publicly available description of C, now known as the K&R standard.

A few information about C:

- C was invented to write an OS called UNIX.

- C is successor of B language introduced in 1970s.

- In 1988 C was formalised by ANSI standard.

- UNIX OS is totally written in C.

- Today C is most popular language and it provides a gateway into the world of programming for beginners and helps them easily understand several other programming languages.
